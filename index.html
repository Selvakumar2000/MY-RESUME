HARDWARE PROJECT:

TEAM SIZE: 3

SMART BINS

OBJECTIVE : To design a system that monitors the level of the bin and alerts the municipality when specified level is reached.

PLATFORM : ARDUINO IDE

COMPONENTS : NodeMcu, Ultrasonic Sensor, Servo Motor, NEO-6M GPS

DESCRIPTION : The idea of the project is to avoid overflow of the bin. The Level of the dustbin is monitored by ultrasonic Sensor. Once the setpoint (say 80%) is reached, the NodeMcu triggers motor to rotate 90 degrees, hence the bin is mechanically locked, simultaneously notification and current location of the dustbin is sent to the Municipality. The Instantaneous level of the bin is displayed in the BLYNK App, also app has a button to open the lock.


SOFTWARE PROJECT:

TEAM SIZE: 2

HOME AUTOMATION USING INTERNET OF THINGS

OBJECTIVE : To create a system that control our home appliances using Google Assistant through voice commands.

PLATFORM : ARDUINO IDE, IFTTT, Adafruit IO, Google Assistant.

COMPONENTS : NodeMcu, Relay, Battery

DESCRIPTION : The Home Appliances are connected to the relay which is controlled by the Nodemcu. The control signal from NodeMcu depends upon the value passed by Adafruit IO. The value passed by the Adafruit IO depends upon the conditional statements set in the IFTTT. The voice commands given to the mobile phone determine the conditional statements and send corresponding data to the Adafruit IO.
